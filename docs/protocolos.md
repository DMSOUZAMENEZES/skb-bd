# Protocolos e Classificacao

Resumo de classificacao para catalogacao no SKB.

## Tipos de Documento

- PCDT: Protocolo Clinico e Diretrizes Terapeuticas
- DDT: Diretriz Diagnostica e Terapeutica
- Diretriz: documento orientador tecnico-assistencial
- Protocolo de Uso: criterio operacional para uso de tecnologia/medicamento

## Regras de Catalogacao

- Identificar tipo documental no momento da ingestao.
- Registrar titulo oficial, URL de origem e data de publicacao (quando disponivel).
- Preservar texto bruto e versao normalizada para busca.
